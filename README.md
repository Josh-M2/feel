# Feel

Guest-first Bible app built with Flutter and Supabase.

## Focus

- Daily personalized verse flow
- Read tab backed by self-hosted scripture content
- Saved highlights, bookmarks, notes, and history
- Reading plans and progress
- Widget-preview and notification foundations

## Project Notes

- App routes and feature structure are defined in `AGENTS.md`.
- Supabase scripture import workflow lives in [supabase/README.md](supabase/README.md).
- The app is actively migrating away from transitional mock and live Bible API paths toward Supabase-hosted scripture as the V1 source of truth.
